# 🇨🇳 中文技能（简体）

[pm-claude-skills](https://github.com/mohitagw15856/pm-claude-skills) 库旗舰技能的专业翻译。**英文版本始终为规范版本**——这些翻译逐节遵循英文版的结构，并在源文件发生大改后同步更新。

| 技能 | 产出什么 |
|---|---|
| [resume](resume/SKILL.md) | 以成就为主线、能通过 ATS 的简历 |
| [executive-summary](executive-summary/SKILL.md) | 3 分钟内可决策的高管摘要 |
| [lease-decoder](lease-decoder/SKILL.md) | 你的租房合同，解读并标出警示信号 |
| [medical-bill-decoder](medical-bill-decoder/SKILL.md) | 医疗账单逐行解读 + 谈判话术 |
| [tos-decoder](tos-decoder/SKILL.md) | 服务条款：你真正在同意什么 |

## 如何使用

和任何技能一样：把文件夹复制到 `~/.claude/skills/`（Claude Code），或把内容作为指令粘贴到 ChatGPT/Gemini，或在[免费演练场](https://mohitagw15856.github.io/pm-claude-skills/)打开英文版。

## 请求或贡献更多翻译

开一个带 `translation` 标签的 issue，或按这些文件夹的模式提交 PR：`name:` 保持英文、`language: zh`、章节结构与源文件完全一致、正文第一行为规范性说明。
