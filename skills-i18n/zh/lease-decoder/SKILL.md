# 租约解读技能

租赁合同是房东一方写的，为房东一方服务。这个技能像一位以读租约为业的犀利朋友那样读它：每个条款是什么意思、哪些会让你真金白银受损、签字之前——而不是之后——该就哪些据理力争。

## 这个技能产出什么

- 一张大白话的逐条解读表
- 按严重程度分级的警示信号，账目算得明明白白（提前退租罚金、自动续约、押金条件）
- 签约前要问房东的问题
- 一份真正可以谈判的条款清单

## 所需输入

仅在未提供时索取：

- **租约文本**——粘贴、照片转录或部分内容均可。就现有内容开展工作，并明确说明哪些标准条款缺失或无法辨认。
- **租金、押金和租期**（若文本中没有）。
- **大致地区**（州/国家）——绝不要猜测；条款的可执行性因地而异。
- 可选：他们最在意什么（宠物、转租、提前退租、在家办公）。

## 框架：严重程度分级

按现实代价给每项发现分级：

- 🔴 **会让你损失真金白银**——自动续约成一个完整新租期、超出重新出租成本的退租罚金、把维修/保养负担转嫁给租户、写来就是让你拿不回的押金返还条件（如"专业清洁"收据）、费用叠加、责任豁免条款。
- 🟡 **不寻常——据理力争**——短时/无通知进入、一刀切的访客限制、强制使用房东指定的保险公司、单方面修改规则、含糊的"超出正常损耗的损坏"。
- 🟢 **标准模板条款**——直说是模板，让读者知道哪些不必担心。

逐项排查租约中的：**自动续约陷阱**（通知窗口；沉默会让你承诺什么）、**维修负担转嫁**、**进入权**（通知期、事由）、**退租条款的账**（算出实际的退出成本）、**押金返还条件**（列出每一项书面条件）。当某条款通常不可执行时（如放弃宜居性保障），标注为：*"通常不可执行——请咨询当地租户组织；可执行性因司法辖区而异。"*绝不把条款无效说成普适事实。

## 输出格式

### 租约解读：[地址或"你的租约"]

**1. 一段话结论**——签 / 先谈判 / 走人，以及原因。

**2. 逐条解读**

| 条款（§） | 条款怎么写 | 对你意味着什么 | 严重程度 |
|---|---|---|---|

**3. 🚩 警示信号，分级排序**——最严重的在前，每条包括：引用的原文、现实中最坏情况的金钱或麻烦代价、以及建议争取的修改。

**4. 退出与押金的账**——提前离开实际要花多少钱，以及拿回押金所附带的每一项条件。

**5. 签约前要问的问题**——3–6 个，按谈判筹码排序。

**6. 什么可以谈**——房东在被要求时通常愿意修改的条款。

在产出物结尾原文附上：*"这是一份大白话解读，不构成法律/财务建议——法律因司法辖区而异；任何关键事项请向合格专业人士确认。"*

## 质量检查

- [ ] 每个警示信号都引用租约原文——条款编号或逐字引用
- [ ] 退租与押金的账用真实数字算出，而非含糊描述
- [ ] 依赖司法辖区的要点均已标注，并附租户组织转介语句
- [ ] 缺失或无法辨认的部分被明确指出，而非敷衍带过
- [ ] 真正的标准条款标记为 🟢，让读者不必被模板吓到
- [ ] 免责声明逐字出现在产出物中

## 反模式

- [ ] 不要虚构文件中不存在的条款——只解读现有内容
- [ ] 不要为显得客观而软化警示信号——会造成真实损失就直说
- [ ] 不要把依赖司法辖区的规则说成普适——标注并转介
- [ ] 不要把所有条款都标成 🔴——全是警报的解读和没有警报一样无用
- [ ] 不要给出诉讼策略或"这是违法的"之类的裁断——那是律师的事

## 依据

租户方租约审查实践——条款分诊、退出成本计算、押金条件审计。
