# 🇯🇵 日本語スキル

[pm-claude-skills](https://github.com/mohitagw15856/pm-claude-skills) ライブラリの旗艦スキルのプロフェッショナル翻訳。**常に英語版が正本です**——これらの翻訳は英語版と同じ構造をセクションごとに踏襲し、ソースの大きな変更後に更新されます。

| スキル | 産出するもの |
|---|---|
| [resume](resume/SKILL.md) | ATS を通過する実績主導の履歴書 |
| [executive-summary](executive-summary/SKILL.md) | 3 分で意思決定できるエグゼクティブサマリー |
| [lease-decoder](lease-decoder/SKILL.md) | 賃貸契約書の解読と危険信号 |
| [medical-bill-decoder](medical-bill-decoder/SKILL.md) | 医療請求書の行ごとの解読 + 交渉台本 |
| [tos-decoder](tos-decoder/SKILL.md) | 利用規約：あなたが本当に同意していること |

## 使い方

他のスキルと同じ：フォルダを `~/.claude/skills/` にコピー（Claude Code）、ChatGPT/Gemini に指示として貼り付け、または[無料プレイグラウンド](https://mohitagw15856.github.io/pm-claude-skills/)で英語版を開く。

## 翻訳のリクエスト・貢献

`translation` ラベル付きの issue を立てるか、これらのフォルダのパターンに従って PR を送ってください：`name:` は英語のまま、`language: ja`、セクション構造はソースと完全に一致、本文の最初の行に正本性の注記。
