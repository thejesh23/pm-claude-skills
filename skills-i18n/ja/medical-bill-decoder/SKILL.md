# 医療請求書解読スキル

医療請求書はコードで書かれている——文字通りの意味で——そして誤りは十分ありふれているので、自分の請求書を注意深く読むことは実際にお金になる。このスキルは各行を翻訳し、おかしく見える請求にフラグを立て、電話で言うべき言葉をそのまま手渡す。

## このスキルが産出するもの

- 請求項目を平易な言葉にした行ごとの解読
- ランク付けされた危険信号：重複請求、アンバンドリング、バランスビリング、あり得ない請求
- 3 つの台本：明細請求書の取得、財務支援の照会、残額の交渉
- 優先順位付きの行動リスト——何をまず、誰に対して争うか

## 必要な入力

提供されていない場合のみ求める：

- **請求書および/または EOB のテキスト**——貼り付けか書き起こし。要約版の請求書しかなければその旨を述べ、明細請求書を求める台本から始める。見えている範囲を解読する。
- **保険の状況**——加入（ネットワーク内/外、分かれば）、未加入、または不明。
- **背景**——受診の理由、および医療機関が緊急時に選ばれたものかどうか。

## フレームワーク：深刻度スケール

すべての発見を評価する：

- 🔴 **実際にお金を失わせ得る**——重複請求、アンバンドリング（1 つの処置を複数の構成コードに分けて請求）、ネットワーク内施設や緊急時のネットワーク外診療に対するバランスビリング、行われていない診療への請求、請求書と EOB の不一致、EOB の「患者負担額」を超える請求。
- 🟡 **異例——押し返す**——大きな金額の曖昧な行（「消耗品」「施設利用料」）、診療レベルのアップコーディングの兆候（単純な受診に最上位の受診コード）、通常を大きく超える請求。
- 🟢 **標準**——自己負担金、免責額の充当、正常に見える項目。そうだとはっきり言う。

コードは対照表ではなく文脈で解読する：CPT/HCPCS や収益コードの*種類*が請求書上の文脈から何を意味するかを説明し、確信を持って解釈できないコードには `[要確認——この項目が何をカバーするか請求部門に質問]` と印を付ける。コードと価格の対応基準を捏造してはならない。請求書と EOB が両方あれば必ず突き合わせる——両者の差こそお金のありかだ。

## 出力形式

### 医療請求書解読：[医療機関 / 受診日]

**1. 結論**——請求総額、正当に見えるもの、争えるもの、現実的な目標金額。

**2. 行ごとの解読**

| 行 / コード | 何と思われるか | 金額 | 評価 | 深刻度 |
|---|---|---|---|---|

**3. 🚩 危険信号、ランク順**——各項目に、該当する行の引用、疑わしい理由、どこに申し立てるか（請求部門、保険会社、または両方）。

**4. あなたの台本**——短い逐語台本を 3 つ：(a) コード付きの完全な明細請求書を求める、(b) 財務支援/チャリティケアと早期支払い割引について尋ねる、(c) 交渉の電話——争点から切り出し、減額と分割払いを求める。すべて書面で確保する。

**5. 行動順序**——番号付きの次のステップ、期限を明記（争っている間に債権回収に回させない——保留を依頼するよう言う）。

成果物の末尾に、一字一句そのまま付す：*「これは平易な言葉での読解であり、法的・財務的助言ではありません——法律は法域により異なります。重要事項は資格を持つ専門家に確認してください。」*

## 品質チェック

- [ ] フラグを立てた請求はすべて請求書上の特定の行を指す。引用または番号で
- [ ] 請求書と EOB が両方あれば突き合わせ済み。不一致が最上位のフラグ
- [ ] 解釈できないコードは `[要確認]` と印を付け、決して診断に推測しない
- [ ] 3 つの台本はすべて一言一句そのまま使える。「言うべきことの要約」ではない
- [ ] バランスビリングのフラグには、保護が法域と保険種別に依存する旨を明記
- [ ] 免責文が一字一句そのまま成果物に現れる

## アンチパターン

- [ ] 文書にない請求、コード、価格を創作しない
- [ ] バランスを装って危険信号を和らげない——重複の疑いは重複の疑いだ
- [ ] 法域依存の請求保護を普遍として提示しない
- [ ] 医療行為そのものを診断・批評しない——解読するのは請求だけ
- [ ] 結果を約束しない（「免除してくれますよ」）——台本は勝算の良い依頼として位置づける

## 基盤

患者側の請求アドボカシー実務——明細請求書監査、EOB 突合、交渉台本作成。
