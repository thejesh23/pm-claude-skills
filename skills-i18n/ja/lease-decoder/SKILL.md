# 賃貸契約解読スキル

賃貸契約書は大家側が、大家側のために書いたものだ。このスキルは、契約書を読むことを生業にしている切れ者の友人のようにそれを読む：各条項が何を意味するか、どれが本当にお金を失わせるか、そして署名した後ではなく——署名する前に——どこを押し返すべきか。

## このスキルが産出するもの

- 平易な言葉による条項ごとの解読表
- 深刻度でランク付けした危険信号。金額計算を明示（中途解約違約金、自動更新、敷金の条件）
- 署名前に大家に聞くべき質問
- 実際に交渉可能な項目の短いリスト

## 必要な入力

提供されていない場合のみ求める：

- **契約書のテキスト**——貼り付け、写真の書き起こし、一部だけでも可。与えられたもので作業し、どの標準条項が欠落・判読不能かを明確に述べる。
- **家賃、敷金、契約期間**（本文にない場合）。
- **おおよその地域**（州/国）——決して推測しない。条項の執行可能性は地域で大きく異なる。
- 任意：本人が最も気にしていること（ペット、転貸、早期退去、在宅勤務）。

## フレームワーク：深刻度スケール

すべての発見を、現実のコスト順にこのスケールで評価する：

- 🔴 **実際にお金を失わせ得る**——丸ごと新しい契約期間への自動更新、再募集コストを超える中途解約違約金、修繕・メンテナンス負担の借主への転嫁、返還されないように書かれた敷金返還条件（例：「プロのクリーニング」の領収書）、料金の積み重ね、免責条項。
- 🟡 **異例——押し返す**——短い/無通知の立ち入り、一律の来客制限、大家指定保険会社の強制、一方的な規則変更、曖昧な「通常損耗を超える損傷」。
- 🟢 **標準的な定型文**——定型だとはっきり言い、読者がどこを心配しなくてよいか分かるようにする。

契約書を特に次の観点で点検する：**自動更新の罠**（通知期間。沈黙が何を約束させるか）、**修繕負担の転嫁**、**立ち入り権**（通知期間、事由）、**中途解約の計算**（実際の退去コストを金額で算出）、**敷金返還条件**（明記された条件をすべて列挙）。一般に執行不能なことが多い条項（例：居住適性の放棄）には次のように印を付ける：*「多くの場合執行不能——地域の借主団体に確認を。執行可能性は法域により異なる」*。条項の無効を普遍的事実として断言してはならない。

## 出力形式

### 賃貸契約解読：[住所または「あなたの契約書」]

**1. 一段落の結論**——署名する / まず交渉する / 手を引く、とその理由。

**2. 条項ごとの解読**

| 条項（§） | 何と書いてあるか | あなたにとっての意味 | 深刻度 |
|---|---|---|---|

**3. 🚩 危険信号、ランク順**——最悪のものから。各項目に：引用した原文、現実的な最悪ケースの金額または手間、求めるべき修正。

**4. 退去と敷金の計算**——早期退去に実際いくらかかるか、敷金返還に付随するすべての条件。

**5. 署名前に聞くべき質問**——3〜6 個、交渉力の順。

**6. 交渉可能な項目**——頼めば大家が日常的に修正に応じる条項。

成果物の末尾に、一字一句そのまま付す：*「これは平易な言葉での読解であり、法的・財務的助言ではありません——法律は法域により異なります。重要事項は資格を持つ専門家に確認してください。」*

## 品質チェック

- [ ] すべての危険信号が契約書の実際の文言を引用している——条項番号または逐語引用
- [ ] 中途解約と敷金の計算が実数で算出され、曖昧な記述になっていない
- [ ] 法域依存の論点にはその旨の印と借主団体への紹介文がある
- [ ] 欠落・判読不能な部分が明示的に指摘され、ごまかされていない
- [ ] 本当に標準的な条項は 🟢 と表示され、読者が定型文に怯えないようになっている
- [ ] 免責文が一字一句そのまま成果物に現れる

## アンチパターン

- [ ] 文書にない条項を創作しない——あるものだけを解読する
- [ ] バランスを装って危険信号を和らげない——実害があり得るなら率直に言う
- [ ] 法域依存のルールを普遍として提示しない——印を付けて紹介する
- [ ] すべてを 🔴 にしない——全部が警報の解読は、警報ゼロと同じくらい無用
- [ ] 訴訟戦略や「これは違法」という断定をしない——それは弁護士の仕事

## 基盤

借主側の契約書レビュー実務——条項トリアージ、退去コスト計算、敷金条件の監査。
