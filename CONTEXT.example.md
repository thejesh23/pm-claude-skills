# CONTEXT.md

> Copy this file to your **project root** as `CONTEXT.md` (or `~/.claude/CONTEXT.md` for a global one), fill it in, and delete the guidance comments. Skills read it so their output comes back in *your* language, about *your* product — no re-typing context every time.
>
> Why this exists: the #1 reason AI output feels generic is the model doesn't know who you are. A short, shared context file fixes that for every skill at once. Keep it tight (half a page beats three pages) and update it when the facts change.
